# 🎯 GATE RAG AI Assistant  

An intelligent **AI-powered Retrieval Augmented Generation (RAG) system**  
designed to help students prepare for **GATE Computer Science (CSE) and Data Science & AI (DA)** exams.

This assistant can answer syllabus queries, analyze trends, generate mock tests, and guide preparation strategy using real **previous year question papers + official syllabus PDFs**.

---

## 🚀 Live Demo  
👉 https://gate-rag-system.streamlit.app  

---

## 🧠 Features  

✅ Retrieval Augmented Generation (RAG) architecture  
✅ Multi-PDF intelligent semantic search  
✅ GATE CSE + GATE DA syllabus understanding  
✅ Previous year question paper trend analysis  
✅ AI mentor style preparation guidance  
✅ Mock test generation support  
✅ Real-time streaming responses  
✅ Clean ChatGPT-style UI  

---

## 🏗️ System Architecture  

1️⃣ PDF syllabus + question papers are ingested  
2️⃣ Text is chunked into semantic segments  
3️⃣ SentenceTransformer embeddings generated  
4️⃣ FAISS vector database built  
5️⃣ User query → semantic retrieval  
6️⃣ Context passed to Groq LLM  
7️⃣ AI generates focused exam-mentor response  

---

## 🛠️ Tech Stack  

- Python  
- Streamlit  
- FAISS (Vector Database)  
- Sentence Transformers  
- Groq LLM API  
- PyPDF2  
- NumPy

## ⚙️ Installation

## 🎓 Use Case

This system acts as an **AI Exam Mentor** that helps GATE aspirants to:

- 📚 Understand important GATE topics quickly  
- 📊 Analyze previous year question paper trends  
- 📝 Generate practice questions and mock tests  
- 🎯 Plan an effective preparation strategy  
- ❓ Clarify syllabus-related doubts using AI
